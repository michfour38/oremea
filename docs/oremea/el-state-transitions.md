\# EL STATE TRANSITIONS



EL is state-driven.



States do not move in a fixed order.



States transition when conditions change.



\---



\# State List



\- unclear

\- choice\_forming

\- resource\_missing

\- resource\_identified

\- possibility\_emerging

\- objection\_present

\- movement\_ready

\- movement\_current



\---



\# Primary Transition Rule



EL moves state only when evidence changes.



Not because time passed.



Not because a question was answered.



Not because a stage completed.



Evidence must change.



\---



\# Unclear → Choice Forming



Condition:



Participant names a desired reality, direction, or outcome.



Example:



"I want Oremea generating $1m+ USD monthly after tax."



\---



\# Choice Forming → Possibility Emerging



Condition:



Participant identifies multiple possible paths.



Example:



"We could use creators, paid ads, partnerships, or affiliates."



\---



\# Possibility Emerging → Choice Forming



Condition:



Participant must narrow.



Example question:



"Which path are you choosing first?"



\---



\# Choice Forming → Resource Missing



Condition:



Participant names a path but cannot move because a resource is absent.



Example:



"I need creators but I don't know who to approach."



\---



\# Resource Missing → Resource Identified



Condition:



The missing resource becomes specific.



Example:



"I need a list of creator prospects."



\---



\# Resource Identified → Movement Ready



Condition:



A resource exists and a next completed action can be named.



Example:



"I created the first creator shortlist."



\---



\# Any State → Objection Present



Condition:



Participant introduces contradiction, disbelief, impossibility, or unresolved concern.



Examples:



"But nobody will respond."



"I don't believe this can work."



"I don't have the money."



"This feels impossible."



\---



\# Objection Present → Previous State



Condition:



Objection is clarified, resolved, reduced, or accepted as manageable.



Example:



"Nobody may respond, but sending the first message is still possible."



\---



\# Movement Ready → Movement Current



Condition:



Reality changes.



A completed action exists.



Example:



"I sent the creator message."



\---



\# Movement Current → Exit



Condition:



Movement is current.



Compass exits.



Reality becomes teacher.



\---



\# State Regression



Participants may regress.



Examples:



Movement Ready → Objection Present



Choice Forming → Unclear



Resource Identified → Resource Missing



Regression is not failure.



Regression is information.



\---



\# State Recalculation



After every response:



EL recalculates state.



EL does not assume the previous state remains correct.



Current evidence wins.



\---



\# State Transition Priority



When several transitions are possible:



Reality Incoherent

↓

Objection Present

↓

Resource Missing

↓

Choice Forming

↓

Possibility Emerging

↓

Movement Ready

↓

Movement Current



Highest-priority unresolved state becomes primary.



\---



\# Final Rule



States are not labels.



States are navigation positions.



EL follows the position.
